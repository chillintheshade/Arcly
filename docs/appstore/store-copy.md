## 描述 (4000 字内)

Arcly 是一款极简而优雅的 macOS 环形启动器。按下快捷键，你最常用的 App 就会以环形排列出现在鼠标周围，一次点击即可启动或切换。不再满屏寻找 Dock，不再让 Spotlight 打断你的思路。
